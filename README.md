## 기능
 - ChatBot이랑 대화하기
	- 대화방 생성 (챗봇용 채널에서 공지사항의 컴포넌트를 통해 스레드 생성)
	- 대화
	- 대화 종료 시 요약본 생성
	- 대화 메시지에 📃 리액션 하면 벡터스토어에 파일 저장
	- 대화 메시지에 📃 리액션을 제거하면 벡터스토어에 저장된 파일 제거
